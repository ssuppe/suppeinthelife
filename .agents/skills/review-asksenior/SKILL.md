# Review Asksenior

Write a high-signal report for a senior engineer to review.

### Content
1. **Problem:** Concise description of what we are solving.
2. **Implementation:** Summary of the approach taken.
3. **Challenges:** Any hurdles or deviations from the original plan.
4. **Hypothesis:** If stuck, explain your latest theory.
5. **Evidence:** Include relevant code snippets and error logs.

Output this as a clean Markdown report to be copy-pasted.
